About This Project

What is Fake News Detection?

This application uses advanced Machine Learning algorithms to identify whether a news article is fake or real. With the rise of misinformation online, this tool helps verify the authenticity of news content.

How It Works
  1. Text Processing: The input text is cleaned and preprocessed using Natural Language Processing (NLP)
  2. Vectorization: TF-IDF (Term Frequency-Inverse Document Frequency) converts text into numerical features
  3. Classification: A Passive Aggressive Classifier trained on labeled fake/real news data makes predictions
  4. Results: The model returns a prediction with high accuracy (95%+)
