# 🚀 QUICK START GUIDE - Fake News Detection App\n\n**For Complete Beginners** ⭐\n\nThis guide will get your app running in **15-20 minutes**!\n\n---\n\n## ✅ Before You Start\n\nMake sure you have:\n- Python installed (download from python.org if not)\n- Git installed (download from git-scm.com if not)\n- A web browser (Chrome, Firefox, Safari, Edge)\n\n---\n\n## 🪟 FOR WINDOWS USERS\n\n### Step 1: Open Command Prompt\nPress `Win + R`, type `cmd`, press Enter\n\n### Step 2: Download Your Project\n```bash\ncd Desktop\ngit clone https://github.com/Deepak-Rithik/fake-news-detection-project\ncd fake-news-detection-project\ngit checkout complete-app-setup\n```\n\n### Step 3: Create & Activate Virtual Environment\n```bash\npython -m venv venv\nvenv\\Scripts\\activate\n```\n\n**You should see `(venv)` at the start of your command line**\n\n### Step 4: Install Libraries\n```bash\npip install -r requirements.txt\n```\n\n**Wait for it to finish** ⏳ (2-3 minutes)\n\n### Step 5: Create Dataset Folder\n```bash\nmkdir model\\database\\model\\model\\dataset\n```\n\n### Step 6: Train the AI Model\n```bash\npython train_model.py\n```\n\n**You should see:**\n```\n✅ Model trained successfully!\n```\n\n### Step 7: Start Your App\n```bash\npython app.py\n```\n\n**You should see:**\n```\n * Running on http://127.0.0.1:5000\n```\n\n### Step 8: Open in Browser\n- Open your web browser (Chrome, Firefox, etc.)\n- Go to: `http://localhost:5000`\n- You should see a beautiful form! 🎉\n\n### Step 9: Test It!\n- Paste some news text\n- Click \"Check News\"\n- See if it's Fake or Real!\n\n### Step 10: Stop the App\nIn the command prompt, press `Ctrl + C`\n\n---\n\n## 🍎 FOR MAC USERS\n\n### Step 1: Open Terminal\nPress `Cmd + Space`, type `terminal`, press Enter\n\n### Step 2: Download Your Project\n```bash\ncd Desktop\ngit clone https://github.com/Deepak-Rithik/fake-news-detection-project\ncd fake-news-detection-project\ngit checkout complete-app-setup\n```\n\n### Step 3: Create & Activate Virtual Environment\n```bash\npython3 -m venv venv\nsource venv/bin/activate\n```\n\n**You should see `(venv)` at the start of your terminal**\n\n### Step 4: Install Libraries\n```bash\npip install -r requirements.txt\n```\n\n**Wait for it to finish** ⏳ (2-3 minutes)\n\n### Step 5: Create Dataset Folder\n```bash\nmkdir -p model/database/model/model/dataset\n```\n\n### Step 6: Train the AI Model\n```bash\npython train_model.py\n```\n\n**You should see:**\n```\n✅ Model trained successfully!\n```\n\n### Step 7: Start Your App\n```bash\npython app.py\n```\n\n**You should see:**\n```\n * Running on http://127.0.0.1:5000\n```\n\n### Step 8: Open in Browser\n- Open Safari, Chrome, or your preferred browser\n- Go to: `http://localhost:5000`\n- You should see a beautiful form! Download from https://www.python.org/downloads/\n\n### Error: \"git is not recognized\"\n**Solution:** Git is not installed. Download from https://git-scm.com/download\n\n### Error: \"ModuleNotFoundError\"\n**Solution:** Make sure virtual environment is activated (see `(venv)` in terminal)\n\n### Error: \"Port 5000 already in use\"\n**Solution:** Another app is using port 5000.
